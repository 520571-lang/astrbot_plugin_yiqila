## 🎭 猎奇功能一览

### 💨 基础排泄指令

| 指令 | 别名 | 功能说明 |
|------|------|----------|
| `/拉屎` | 拉粑粑、拉臭臭、开大了 | **开启你的排便之旅！** 记录神圣时刻，向全群宣告：爷要开始产出了！ |
| `/一起拉` | 查看名单、谁在拉、偷窥厕所 | **偷窥群友的肠道状态！** 看看谁正在马桶上和你隔空对视 👀 |
| `/拉完了` | 拉完、结束拉屎、收工了 | **宣告战役胜利！** 记录你的排便时长，成为群里的"持久王" |
| `/便秘了` | 卡住了、出不来 | **紧急求救信号！** 召唤群友的鼓励和精神支持 💪 |
